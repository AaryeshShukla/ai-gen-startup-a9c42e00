# Jersey Startup Landing Page

## Overview
This is a single-page web application for a startup selling jerseys. It features a modern dark theme with neon gradients, a sticky navigation bar, an animated hero section, interactive features, a pricing comparison table, customer reviews, and an FAQ section.

## Features
- **Responsive Design**: The layout adapts to various screen sizes.
- **Email Capture Form**: Users can subscribe to updates.
- **Interactive Cards**: Features section with hover effects.
- **Pricing Plans**: Monthly and annual pricing options.
- **Customer Reviews Slider**: Displays customer feedback.
- **FAQ Accordion**: Interactive questions and answers.

## Setup Instructions
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Enjoy the interactive features and modern design!
